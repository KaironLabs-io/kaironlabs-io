# Kairon Labs

**Building AI tools that respect your time and your data.**

Kairon Labs develops AI-powered tools for developers and teams — web chat, desktop agents, a code assistant, and a developer API — all backed by private, self-hosted language models.

### Products

- **Kairo** — AI chat platform with image generation and project workspaces
- **Kairo Desktop** — Local AI agent for Windows, macOS, and Linux
- **Kairo Code** — AI-powered code assistant and CLI tool
- **Kairo API** — OpenAI-compatible REST API for developers
- **Kairo Agents** — Build and deploy autonomous AI agents with a simple SDK

### Models

Our model family — Nyx, Theron, and Helios — is designed for real work. Fast, private, and self-hosted.

### Enterprise

Custom AI chatbots, dedicated model hosting, and private deployments for organizations that need control and compliance.
